# 📊 Sales Dashboard - Power BI Project

## 📌 Project Overview
This project focuses on analyzing sales data and creating an interactive dashboard using Power BI. The dashboard provides insights into sales performance, customer behavior, and regional trends.

---

## 🎯 Objectives
- Analyze sales performance across different regions
- Identify top-performing products and categories
- Understand customer purchasing patterns
- Provide insights for business decision-making

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Microsoft Excel (Dataset)

---

## 📊 Key Features
- Interactive dashboard with filters and slicers
- KPIs for revenue, profit, and sales trends
- Region-wise and category-wise analysis
- Dynamic visualizations for better insights

---

## 🔍 Data Cleaning & Transformation
- Removed duplicates and handled missing values
- Standardized data formats
- Created calculated columns and measures using DAX

---

## 📈 Insights
- Identified top-performing products contributing to revenue
- Found key regions driving maximum sales
- Analyzed customer trends for better targeting

---

## 📂 Files Included
- Sales Dashboard.pbix (Power BI file)
- Dataset used for analysis

---

This project demonstrates my ability to clean, analyze, and visualize data using Power BI to generate actionable insights.
